# 🎮 Three.js 3D 抽卡系统

## 📌 项目简介
本项目是一个基于 Three.js 的 3D 抽卡系统，模拟主流游戏（如原神）的抽卡体验，实现了从 UI 到渲染管线的完整设计。

---

## 🚀 技术亮点

- Three.js 3D 渲染
- ShaderMaterial（金光流动特效）
- EffectComposer 后处理
- UnrealBloomPass 泛光效果
- 分层 Bloom（仅 SSR 发光）
- 抽卡概率 + 保底机制

---

## 🎬 核心效果

- ⭐ 星空背景（3D）
- ⭐ SSR 金光爆发
- ⭐ 卡牌展示
- ⭐ 分层发光
